WebStorm激活（2017.3.3）

最新版本的WebStorm更新后，无法再使用网上的那些license server ，activation code也是激活后立即失效。查到的新方法参考lan yu博客。

步骤如下：

  1. 下载破解文件
    链接: https://pan.baidu.com/s/1dFS9DaL
    密码: xxc2
  2. 找到对应的系统版本，mac系统对应的是IntelliJIDEALicenseServer_darwin_amd64。双击打开运行。
  3. 打开WebStorm，Help -> Register -> License Server，填写License Server，为 http://127.0.0.1:1027 ,点击Activate即可激活

注意：

  1. 此方法只验证mac版本，其他系统不能保证一定有效。
  2. 参考资料：lan yu博客
  3. lanyu博客中也提供了反向代理工具，但尝试后没有成功，各位也可以试试。

使用DeDRM去除kindle的DRM保护

下载软件

  1. 通常使用calibre来管理kindle书库
  2. calibre用于破解DeDRM的插件——DeDRM，认准这家，网上其他插件可能我用的方法不对或者已经失效了，没有可用的。此插件可以单独使用，也可以和calibre结合，本文只针对使用calibre的情况。如下所示：

    注：下载之后不要解压，保留压缩文件供calibre使用

使用步骤

  1. Calibre -> Preferences -> 插件（高级选项中）-> 从文件加载插件（右下角），会报一些警告，忽略即可。如图：

  2. 双击插件，打开选项 ‘eInk Kindle ebooks’，添加自己kindle的序列号，是必须的，否则破解DRM的时候会报错。序列号在kindle上，查看方法请自行百度。

  3. 从亚马逊下载书籍，在”管理我的内容和设备”中，点击左侧的三个点的按钮，下载书籍到电脑上。

  4. 把书导入calibre，即可自动破解。然后发送到kindle/邮箱即可。
